How to use tags

  •  Creating a blog
  • Example of some org syntax 

Tags are useful to classify o-blog articles/posts (but not for pages). They are stored as Org-mode tags for each entry. To add or modify a tag for a post, just use C-c C-c when on an Org-mode headline, and enter the tag value.

If an article has more than one tag, separate them by a colon (:).

Special characters used in tags:

  • At sign (@) is converted to a dash (-).
  • Underscore (_) is converted to a blank ( ).
