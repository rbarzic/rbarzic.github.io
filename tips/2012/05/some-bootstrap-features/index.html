Using labels

Label and annotate text using a label:TYPE link prefix where TYPE is one of default, success, warning, important, info or reverse. The URL part of the link is used as the label text. 

#+BEGIN_HTML
 <span class="label label-default">Default</span>
#+END_HTML

#+BEGIN_HTML
 <span class="label label-success">Success</span>
#+END_HTML

#+BEGIN_HTML
 <span class="label label-warning">Warning</span>
#+END_HTML

#+BEGIN_HTML
 <span class="label label-important">Important</span>
#+END_HTML

#+BEGIN_HTML
 <span class="label label-info">Info</span>
#+END_HTML

#+BEGIN_HTML
 <span class="label label-inverse">Inverse</span>
#+END_HTML
Default Success Warning Important Info Inverse

Using badges

Indicators and unread counts as for labels. Use the badge:TYPE link prefix. 

#+BEGIN_HTML
 <span class="badge badge-default">Default</span>
#+END_HTML

#+BEGIN_HTML
 <span class="badge badge-success">Success</span>
#+END_HTML

#+BEGIN_HTML
 <span class="badge badge-warning">Warning</span>
#+END_HTML

#+BEGIN_HTML
 <span class="badge badge-important">Important</span>
#+END_HTML

#+BEGIN_HTML
 <span class="badge badge-info">Info</span>
#+END_HTML

#+BEGIN_HTML
 <span class="badge badge-inverse">Inverse</span>
#+END_HTML
Default Success Warning Important Info Inverse

Using progress bars

For loading, redirecting, or action status. Use a progress:TYPE link where TYPE is one of info, success, warning or danger. Additional classes (striped or active) can be added using comma. The URL is the progress percent. 

#+BEGIN_HTML
 <div class="progress progress-info "><div class="bar" style="width: 25%;"></div></div>
#+END_HTML

#+BEGIN_HTML
 <div class="progress progress-success progress-striped"><div class="bar" style="width: 50%;"></div></div>
#+END_HTML

#+BEGIN_HTML
 <div class="progress progress-warning progress-striped active"><div class="bar" style="width: 75%;"></div></div>
#+END_HTML

#+BEGIN_HTML
 <div class="progress progress-danger "><div class="bar" style="width: 100%;"></div></div>
#+END_HTML
